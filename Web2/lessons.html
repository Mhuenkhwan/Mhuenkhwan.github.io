การใช้งาน specificity

Selector HTML CSS
Element div or nav or others div{} or nav{} or others
ID id="id-name" #id-name{}
CLASS class="class-name" .class-name{}
Selector มีอยู่หลายแบบ การเลือกใช้งานจึงแตกต่างกัน
- Element Selector ชื่อของ selector จะเป็นชื่อเดียวกับ Tag Name ของ html และจะมีผลกับทุก Element ที่มี Tag Name เดียวกับ Selector

ผลลัพธ์

- ID Selector ใช้กำหนดรูปแบบเฉพาะ ตั้งชื่อได้ตามต้องการ แต่ห้ามนำหน้าด้วยตัวเลข และเว้นวรรคให้แทนด้วย - ใช้ได้กับ element เดียว และ element นั้นจะต้องใช้ id ได้ id เดียวเท่านั้น

ผลลัพธ์

- CLASS Selector ใช้กำหนดรูปแบบเฉพาะ ตั้งชื่อได้ตามต้องการ แต่ห้ามนำหน้าด้วยตัวเลข และเว้นวรรคให้แทนด้วย - ใช้ได้หลาย element และ element นั้นใช้ได้หลาย class
